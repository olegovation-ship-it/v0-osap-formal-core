
# Draft Release Notes - V0 OSAP v1.3.0-rc1

**DRAFT ONLY - TAGGING NOT AUTHORIZED BY THIS PATCH**

The candidate consolidates the accepted development theorem corpus T121-T156,
including the normative T121-T150 range and the explicit post-v1.1 extension
T151-T156. It includes executable Python validation, JSON Schema support,
positive and countermodel fixtures, Lean 4 and Coq mappings, canonical theorem
crosswalks, historical-preservation evidence, and RC1 gate auditing.

The candidate does not claim checker completeness, proof-term identity,
unconditional global soundness, global conservativity, or empirical/physical
validation. T150 and T156 remain conditional. The historical v1.2.0 release and
DOI `10.5281/zenodo.21306969` remain immutable.
