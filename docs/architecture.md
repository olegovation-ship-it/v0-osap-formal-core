# Architecture

```text
JSON input
  └─ Draft 2020-12 schema validation
       └─ canonical in-memory representation
            └─ FC-1 semantic checks
                 ├─ identifiers and references
                 ├─ token partition / guard discipline
                 ├─ prerequisites
                 ├─ DLE
                 ├─ absolute-relative firewall
                 └─ observer certification limit
                      └─ deterministic diagnostics
```

The Python checker is executable evidence at E2 only after fixture replay. Lean 4 and Coq are independent theorem-facing mappings. Cross-assistant parity is a separate E5 obligation.
